Fixing some popular smartphone and laptop related issues

There are various issues that come up when one is using tech, be it a smart watch, smart TV, laptop, smartphone and even ironically smart Home devices like Fridges.
As many as they are, I will talk about one of these issues that almost every laptop user has encountered and other issues will be discussed on provided articles.
Okay, let's be honest. Windows users have seen this a couple of times, others more than the fingers on your right hand.
After seeing this, everyone has that panic strike that kicks in, thinking OMG, my laptop is broken. After getting it to a repair shop you're told "just leave it here and we'll fix it for you".
They don't do any magic, they just take a few steps that you can take as well to fix it. Remember, do everything carefully or you will end up crying alone.

  1. Once you see the screen reading(Automatic Repair), there are usually 2 options:
    • Restart
    • Advanced Options
    Usually, people tend to click on restart, running into the same screen again, or click on advanced options and try to reset the pc.
    Do this instead, once you click on advanced options, it will have several options. Select command prompt
  2. After selecting command prompt, it may ask for user and password, you already have those. On the command prompt window that shows up, type Diskpart.
    Diskpart is a disk optimization tool used in doing certain operations to your drives/disks
  3. Afterwards type List disk, listing all the available disks in that laptop meaning all your flash drives if connected, all your external and internal drives will be listed, alongside their drive letters.
    Usually windows is installed on drive C, but if you had renamed it, it will still read whatever you had named it.
    I know you are asking what we are looking for. Most of the times a disk may fail or "crash", which means accessing it's information like files or Operating systems will be difficult. We want to see if your disk containing your Windows Operating System is still there and alive. Which you'll know if it reads it's original size, whether 256GB, 500GB, whichever, that will show that it is alive.
    If it doesn't read your disk at all it means you have a faulty connection where your disk is connected, that just means it shifted a little bit. If it reads but shows a false storage size like 6778Bytes, well that is direct that it crashed.
  4. Now let us check if Windows, whether Windows 11 or 10, is still there or it crashed. I know what you are asking and yes, an operating system can crash.
    After typing list disk, type list volume, which lists the volumes, this is where you might find the letter c.
  5. Assuming it is letter C that has your windows OS, you will exit Diskpart by typing exit then typing dir C:\Windows(Check the spelling and cases). This will check if windows is on that drive. If windows is there, there will be a list that is displayed of the contents there. If not then Windows decided to say bye bye and you will need a fresh OS installation.
  6. If everything was a success then you just need to restart your pc, and it should boot into windows directly without any problems. If it still loops on that spot, you may need to shutdown your pc for a while before using it again. Sometimes devices may buffer and show erros that are not there.
  7. Lastly, make sure you keep your pc always up to date and research on Tech issues or reach us Grinch or MaRval Technologies for more guided assistance.
